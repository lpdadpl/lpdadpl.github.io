# Prueba.github.io

Este proyecto es una página web de ejemplo creada para pruebas y demostraciones.

   ```bash
   git clone https://github.com/lpdadpl/Prueba.github.io.git
   ```
2. Abre el archivo `main.html` en tu navegador.
